MAX Stochastic objects

_prob_

standard MAX external

prob creates a dynamic first-order Markov chain in response to incoming lists of transition probabilities in the form: state n, state n+1, weight. New lists can be added while a patch is running, allowing the Markov chain to evolve over time. Each incoming bang causes prob to send a new state out the left outlet based on the current state and the stored transition table. If no successor to the current state has been defined, prob sends a bang out the right outlet, possibly signalling the conclusion of a phrase or section. prob also responds to a reset message, which makes it jump to a specified state if it can't find a successor to the current state. At any time, the entire transition table can be displayed in the MAX window.

_anal_

standard MAX external

anal is a companion to prob used to create transition tables. Each time an int is received at the inlet, prob reports the number of times it and the preceeding int have ocurred as a pair. The output is in the form of a list: int n-1, int n, weight, which can be sent directly into prob.

_histo_

standard MAX external

histo is used to create a histogram of incoming ints 0-127. Each time an int is received, its corresponding register is incremented. histo can be connected directly to the table object, for graphical display of the histogram, and also to take advantage of table's probability functions.
